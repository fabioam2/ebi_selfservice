As credenciais do ambiente de teste são definidas pelo administrador. Consulte o responsável pela configuração local.

🛡️ Testes rápidos de segurança

  • Tentar ler config.ini → deve retornar 403
  • Tentar ler cadastro_criancas.txt → 403
  • Path traversal no preview_backup → 404 sem vazar caminho
  • Tentar ler admin_users.json → 403
  • Tentar ler arquivo oculto → 403
